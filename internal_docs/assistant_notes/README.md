# Assistant notes (internal)

Эта папка — рабочие заметки в рамках проекта.

Состав:
- `dev_log.md` — хронология изменений/решений (как журнал разработки).
- `impl_notes.md` — объяснение «как работает/как реализовано» в компактном справочном виде.

Примечание: это *внутренняя* документация для ускорения работы над проектом. Ключевые выводы и итоговые решения дублируются в публичных `README.md` сервисов и в `docs/step-*.md`.
