# Заметки ассистента (внутреннее)

Эта папка - внутренние заметки для ускорения разработки. Здесь мы фиксируем:

- `dev_log.md` - хронология изменений (что сделано/почему/где лежит).
- `impl_notes.md` - подробная карта реализации: как работают потоки и где искать код.
- `Persona.md` - портрет агентной персоны и стилистика ответов приложения.

Правило: любые решения и механики должны отражаться в `impl_notes.md`
и кратко упоминаться в `dev_log.md`.
